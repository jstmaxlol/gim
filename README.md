# gim
`gim` (_`g` interactive mode_) is the interactive REPL for [`g`](https://github.com/jstmaxlol/g/), the *permissive* git wrapper.

## to go to the `gim`
you can either;
* get it from the AUR: `paru -S gim`
* clone the repo and build `gim` manually.
