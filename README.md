# gim
**g** **i**nteractive **m**ode is the interactive REPL for [**g**](https://github.com/jstmaxlol/g/), the \*permissive\* [git](https://git-scm.com) wrapper. \
thanks to **gim** you are able to:
* code shit. (with [vi](https://ex-vi.sourceforge.net/), [vim](https://vim.org/) or [neovim](https://neovim.io/))
* manage your **git** workflow, aka, do **git** stuff.

without ever needing to leave **gim**.

&nbsp;
also, yes the name **gim** is a [**vim**](https://vim.org) joke.

## to get `gim`
you can either;
* install the [AUR](https://aur.archlinux.org/packages/gim) package: `paru -S gim`
* clone the repo and build `gim` manually.
